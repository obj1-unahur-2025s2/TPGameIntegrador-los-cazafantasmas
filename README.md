## Reglas de Juego / Instrucciones

Sos un fantasma y tu mision es asustar a todos los integrantes de la mansion sin que te atrapen los cazafantasmas que buscan matarte.
cuanto mas asustes mas puntos ganas y podes mejorar tu sustos para ganar mayor puntaje.
ademas podes ganar inmunidad para que no te maten durante un tiempo o recuperar vida.
